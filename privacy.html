Политика конфиденциальности

Последнее обновление: 9 июня 2026 г.

1. Кто мы

AD Pulse — сервис автоматического учёта материалов для производственных компаний. Разработан и поддерживается ТОО «Altai Dynamics» (Казахстан).

2. Какие данные мы собираем

  • Аккаунт: email, имя, должность, компания
  • Производственные данные: материалы, транзакции, производственные планы — данные вашей компании
  • WhatsApp-сообщения: входящие сообщения через Wazzup API для автоматического парсинга транзакций
  • Технические данные: IP-адрес, браузер, логи ошибок

3. Как мы используем данные

  • Предоставление функционала платформы (учёт, отчёты, планы)
  • AI-парсинг сообщений WhatsApp через OpenAI API (тексты сообщений передаются для распознавания)
  • Отправка Telegram-уведомлений по вашим настройкам
  • Отправка системных email (подтверждение, сброс пароля, приглашения)
  • Улучшение качества сервиса

4. Передача данных третьим сторонам

Мы передаём данные только следующим сервисам, необходимым для работы платформы:

  • Supabase — база данных и аутентификация (EU регион)
  • Vercel — хостинг приложения
  • OpenAI — AI-парсинг текстов (только содержимое сообщений, без персональных данных)
  • Wazzup — интеграция WhatsApp
  • Resend — транзакционные email

Мы не продаём и не передаём ваши данные третьим лицам в маркетинговых целях.

5. Хранение данных

Данные хранятся на серверах Supabase (Европейский союз). Период хранения — в течение срока действия аккаунта и 30 дней после его удаления.

6. Ваши права

  • Запрос копии ваших данных
  • Исправление неточных данных
  • Удаление аккаунта и данных
  • Возражение против обработки

Для реализации прав напишите на: uskembayev.zh@altaidynamics.kz

7. Безопасность

Мы используем шифрование HTTPS, изоляцию данных по компаниям (мультитенантность), Row-Level Security в базе данных Supabase. Пароли не хранятся в открытом виде.

8. Cookies

Мы используем сессионные cookies для аутентификации (Supabase Auth). Маркетинговые или аналитические cookies не используются.

9. Изменения политики

При существенных изменениях мы уведомим вас по email. Актуальная версия всегда доступна на этой странице.
